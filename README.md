# MSBuild Seoul

Static GitHub Pages site for Microsoft Build //localhost:Seoul.

## Pages

- `index.html`: full landing page
- `embed.html`: compact embeddable event page

## Speaker Images

Replace the placeholder SVGs in `assets/speakers/` with the provided speaker photos using the same file names:

- `park-gusam.svg`: 박구삼
- `kim-jinseok.svg`: 김진석
- `justin-yoo.svg`: 유저스틴
- `lee-jongin.svg`: 이종인

After replacing them, update the file extensions in `index.html` and `embed.html` if you use `.jpg`, `.png`, or `.webp`.
